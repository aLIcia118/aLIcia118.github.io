Web 2.0

Web 2.0 also known as the "Second generation of the World Wide Web" is an improved version of web 1.0

Where there is a shift in how society is able to interact with the internet.

The main differences is with Social Media where individuals are able to connect and share content.

The new implementations Web 2.0 has is a more interactive interface where individuals are able to
collaborate and share information through technology.
Shifting from a read-only web (web 1.0) in web 2.0 individuals are able to read and write web.
Many of the functions Web 2.0 has are still seen in our internet today, as it continues to shape
our technological development and experiences.
